## Intro
This packege allows you to play 5 console games. 

Node.js version > 13  is required.

## Install

```bash
$ npm install @hexlet/code
```
## Start games

```bash
$ brain-even

$ brain-calc

$ brain-gcd

$ brain-progression

$ brain-even
```
